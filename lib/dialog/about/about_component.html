About Notepad 8080 v0.0.15

np8080.win is a web based text editor.

Your data is saved in your web browser's Local Storage. It is NOT stored on any server.

Click the 'Download' button to store the current contents on the computers disk.

This app is written with Dart and AngularDart. Read about it on the 'Diving into Dart' blog!

The source code is available from GitHub.
